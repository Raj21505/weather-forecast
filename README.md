A real-time Weather Application built using Python and Streamlit.
It fetches live weather data using OpenWeatherMap API.

## Tech Stack
- Python
- Streamlit
- OpenWeatherMap API
- Requests

## Features
- Search weather by city
- Temperature, humidity, wind speed
- Modern UI
- Real-time data

## How to Run
1. Clone the repository
2. Install dependencies
3. Add API key in .env
4. Run `streamlit run app.py`
